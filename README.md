## Philosophy

This repository emphasizes:
- **Intuition over formalism** - Understanding *why* before *how*
- **Visual thinking** - Using analogies and mental models
- **Conceptual clarity** - Building from first principles
- **Wonder** - Maintaining the sense of awe at quantum phenomena
